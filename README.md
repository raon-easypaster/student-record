# 🎓 학교생활기록부(생기부) 통합 관리 및 대입 진로 분석 플랫폼

고등학교 1학년부터 3학년까지 3년간 한 명의 학생이 자신의 학업 성취(내신/모의고사)를 분석하고, 창체 및 세특 활동을 누적 기록하며, 대입 희망 대학/학과의 전형 조건과 학생부 데이터를 Gemini AI로 교차 분석하여 개인별 보완 전략을 수립하는 **3개년 통합 진로 설계 관리 플랫폼**입니다.

---

## 🌟 주요 핵심 기능

### 1. 종합 대시보드 (홈)
* **D-Day 타이머**: 학년에 따라 자동으로 졸업년도 대입 수능 예정일까지 남은 디데이를 계산하여 노출합니다.
* **현황 요약**: 최근 기록한 생기부 활동 목록, 아카이빙 자료, 목표 달성률 게이지 및 1순위 목표 대학의 최신 AI 매칭 수준을 한눈에 표시합니다.
* **프로필 편집**: 우측 상단 위젯에서 희망 진로 및 특기 메모를 실시간으로 간편히 편집해 저장할 수 있습니다.

### 2. 성적 관리 (`Academic`)
* **내신 성적 탭**: 중간/기말/학기말고사별 과목, 등급, 이수단위, 원점수, 표준편차를 테이블 형태로 등록합니다. 이수단위 가중치를 적용한 종합 GPA 평점이 실시간 자동 연산됩니다.
* **모의고사 탭**: 학평/모평별 국어, 수학, 영어, 탐구 영역의 등급/표점/백분위를 입력하고 전체 평균 등급을 즉시 확인합니다.

### 3. 생기부 활동 기록 (`Activities`)
* **카테고리 분류**: 자율 / 동아리 / 진로 / 봉사 / 독서 / 세특 / 행특의 7대 생기부 영역에 맞춰 상세 활동 사실과 성찰 회고록을 기록합니다.
* **모아보기 토글**: "학년별 보기(3개년 타임라인 정렬)" 및 "유형별 보기(탭 방식 필터)" 모드를 지원합니다.
* **자료 유기적 연동**: 자료실에 보관된 참고 파일 및 링크를 활동 카드에 다대다로 즉시 태깅하여 구조적으로 연결할 수 있습니다.

### 4. 학년별 성장 추이 (`Growth`)
* **내신 등급 곡선**: Recharts 기반으로 Y축을 등급 역순(1등급이 최상단)으로 표현하여 성적 상승 곡선을 시각화합니다.
* **모의고사 종합 차트**: 등급(좌축)과 백분위(우축) 변화를 이중 축 라인 그래프로 대조합니다.
* **활동 밸런스 분석**: 학기별 7대 영역 누적 막대 차트를 제공하여 활동 편중도를 교정할 수 있게 돕습니다.
* **통합 타임라인**: 성적, 모의고사, 생기부 기록이 모두 결합된 3개년 수직 폭포수 타임라인을 제공합니다.

### 5. 학기별 목표 관리 (`Goals`)
* 매 학기 실천 과제를 설정하고 진행 중 / 달성 / 미달성 상태를 체크합니다.
* 전체 6개 학기의 목표 수행률 추이를 게이지 바 형태로 모아보며 자기주도적 성장을 도모하고 성찰 메모를 기록합니다.

### 6. 대학·학과 분석 (`University`)
* **입학요강 웹 검색**: 희망 대학과 학과를 검색하면 입학처 수시 가이드를 웹에서 크롤링 및 검색해 요약하고 공식 출처 URL을 명시합니다.
* **Gemini AI 적합성 분석**: 학생의 3개년 누적 내신, 모의고사, 창체 활동 내용을 기반으로 전형 인재상 매칭 스코어를 산정하고 경쟁력(강점) 및 보완이 시급한 구체적인 실천 과제를 제시합니다.
* **다차원 대조표**: 분석을 거친 여러 대학 및 학과 스펙을 한 표에 나열하여 최선의 원서 전략 수립을 지원합니다.

### 7. 참고 자료실 (`Library`)
* 교과 탐구 및 소논문, 발표 준비를 위해 수집한 로컬 파일(PDF/압축 등)을 Supabase Storage에 보관하거나, 일반 웹 주소 및 유튜브 링크를 아카이빙합니다.
* 링크 등록 시 Gemini AI가 본문 정보를 자동 독해하여 **3줄 핵심 요약**을 실시간 생성합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Frontend**: React (v19) + TypeScript + Vite + TailwindCSS
* **Database & Auth & Storage**: Supabase (Postgres)
* **AI Engine**: Google Gemini 1.5 Flash API (MIME Type JSON 분석)
* **Data Visualization**: Recharts (v2)
* **Icons**: Lucide React

## 🚀 시작하기 (Quick Start)

### 1. 패키지 설치
```bash
npm install
```

### 2. 로컬 개발 서버 시작
```bash
npm run dev
```
기본적으로 브라우저가 열리며 [http://localhost:5173](http://localhost:5173)에서 앱이 시작됩니다.

### 3. macOS 바탕화면 바로가기 활용
* **`생기부_마스터_실행.command`**: 바탕화면의 해당 파일을 더블 클릭하면 자동으로 로컬 개발 서버가 구동되고 브라우저 탭까지 띄워줍니다.
* **`생기부_마스터_바로가기.webloc`**: 이미 서버가 구동 중인 상태에서 곧바로 접속할 수 있게 열어주는 웹 링크 바로가기입니다.

## ⚙️ 환경 변수 설정 (.env)

프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 아래와 같이 Supabase 및 Gemini 키 정보를 입력하십시오.

```properties
# Supabase URL 및 Anon Key
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# Google Gemini API Key
VITE_GEMINI_API_KEY=your_gemini_api_key
```

> **Note (체험용 데모 모드)**:
> 해당 API 키를 입력하지 않은 상태에서도 로그인 페이지 하단의 **"체험용 데모 계정으로 시작하기"** 버튼을 통해 기재 완료된 1~2학년 샘플 성적 및 활동 기록으로 모든 기능을 100% 모의 구동해 볼 수 있습니다.
